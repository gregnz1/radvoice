# Product

## One-Line Pitch

An iPhone and web dictation workspace that lets a radiologist speak in fragments while an LLM continuously formats the material into a consistent, editable report draft.

## Primary User

A reporting radiologist who wants faster, cleaner reporting without fighting rigid dictation commands or manually reorganizing stream-of-consciousness observations.

## Core Workflow

1. Open a report session on the web or iPhone.
2. Select or auto-detect the examination type.
3. Dictate naturally while reviewing images.
4. The raw transcript syncs immediately.
5. The formatted report updates in near-real time.
6. The system flags ambiguity, contradiction, missing laterality, missing measurements, and unsupported generated text.
7. The radiologist edits and approves the draft.
8. The final text is copied into the RIS.

## Experience Goals

- Fast enough that it feels like live dictation, not batch processing.
- Quiet interface suited to repeated clinical use.
- Clear separation between what was spoken and what was generated.
- Minimal clicks during active reporting.
- Personal style preserved through templates and phrase preferences.

## Non-Goals For MVP

- Automatic diagnosis.
- Direct submission into RIS.
- Replacing radiologist review.
- Full PACS/RIS integration.
- Autonomous clinical decision-making.

## Differentiators

- Fragment-tolerant reporting.
- Radiology-specific templates rather than generic medical summarization.
- Raw transcript and draft report shown together.
- Quality checks designed around reporting errors.
- Web handoff optimized for copy/paste into existing systems.
