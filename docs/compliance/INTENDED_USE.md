# Intended Use

RadVoice is a clinician-controlled dictation, transcription, formatting, and text-transfer tool for radiology report drafting.

RadVoice is a private evaluation prototype. It is not ready for clinical deployment, public App Store distribution, or commercial supply without formal privacy, security, legal, and regulatory review.

RadVoice does not:

- accept patient identifiers
- store patient information
- interpret medical images
- make diagnoses
- recommend findings
- autonomously finalize reports
- submit reports directly to a RIS

The radiologist remains responsible for reviewing, editing, approving, and transferring final report text into the clinical system of record.

## Intended User

Qualified radiologists and authorized clinical users who are creating their own draft report text.

## Intended Workflow

1. The user dictates findings without patient identifiers.
2. RadVoice transcribes and formats the dictated text.
3. The user reviews the draft report.
4. The user edits and explicitly approves the final text.
5. The user manually copies approved text into the RIS.

## Product Boundary

RadVoice is not intended to provide clinical decision support, image interpretation, triage, diagnostic recommendations, or autonomous reporting.

Safety flags are conservative workflow warnings. They are not diagnoses, recommendations, or substitutes for radiologist judgement.
